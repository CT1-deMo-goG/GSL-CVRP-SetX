# GSL Engine: Set X Performance Portfolio
**High-Velocity Deterministic Solver for CVRP (100 - 1,000 Nodes)**

## Overview
GSL Engine is a proprietary, high-performance solver designed for the Capacitated Vehicle Routing Problem (CVRP). It utilizes a **Deterministic Structured Framework** to deliver stable, reproducible, and **Real-time** solutions for industrial logistics.

---

## 📊 Benchmark Validation (Set X)
Evaluated against all 100 instances of CVRPLIB Set X. Our benchmarks compare GSL Engine's deterministic performance against standard heuristic baselines: **Clarke-Wright (CW)** and **Large Neighborhood Search (LNS - 300 Iterations)**.

### Performance Summary
* **Instances:** 100 / 100 (Full Coverage)
* **Feasibility:** 100% Guaranteed Feasible Routes
* **Win Rate vs LNS (300 Iter):** 71% (GSL provides superior speed and stability in the majority of complex topologies)
* **Hardware:** Verified on Mobile Snapdragon Architecture (via Pydroid 3)
